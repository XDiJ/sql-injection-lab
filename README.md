# 🛡️ Python Tkinter SQL Injection Lab

An educational lab demonstrating critical SQL injection vulnerabilities in a desktop application. This project uses Python, Tkinter for the GUI, and MySQL for the backend database. 

> **⚠️ Disclaimer:** This project is intentionally vulnerable and created strictly for educational purposes and cybersecurity awareness. Do not use this code in a production environment.

## 🚀 Project Overview
This application simulates a simple login screen that directly concatenates user input into SQL queries. It provides a hands-on environment to understand how SQL injection attacks work and why parameterized queries are essential for secure software development.

## 🛠️ Prerequisites
To run this lab locally, you will need:
* Python 3.x
* MySQL Server
* `mysql-connector-python` library (`pip install mysql-connector-python`)
